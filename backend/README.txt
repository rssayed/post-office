How to install:

1. Install Python 3.9.4 on your machine
2. On VSCode open a command prompt terminal
3. Enter 'cd backend'
4. Enter 'pip install -r requirements.txt' to have all required packages installed
5. Enter 'venv\scripts\activate' to start the virtual environment
5. Enter 'set flask_env=development'
6. Enter 'flask run'
